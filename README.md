PIN Configuration 🐱‍👤

SD CARD 🤠 

|CS = 17 gpio|

|MOSI = 19 gpio|

|MISO = 16 gpio|

|SCK = 18 gpio|

|GND = GND|

|VCC = 3.3v|
_________________________________
BUTTONs ♟

|UPBUTTON = 15 gpio|

|DOWNBUTTON = 14 gpio|

|SELECTBUTTON = 13 gpio|

|BACKBUTTON = 12 gpio|
_________________________________
IR LED  🏴‍☠️

|Positive = 3.3v OR 5v|

|SIG = 11 gpio|

|GND = GND|
_________________________________
RTC MODULE 🧭

|SCL = 1 gpio|

|GND = GND|

|VCC = VCC|
